# Placeholder untuk Icon Extension

Silakan tambahkan icon extension dengan nama file berikut di folder `public/images/`:

- `icon16.png` - Icon ukuran 16x16 pixel
- `icon48.png` - Icon ukuran 48x48 pixel
- `icon128.png` - Icon ukuran 128x128 pixel

Icon ini akan digunakan pada toolbar browser dan halaman extension management.

Anda dapat membuat icon menggunakan tools seperti Figma, Canva, atau Photoshop.
